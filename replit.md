# Simple Bonjour Page

## Overview
A minimal HTML page that displays "Bonjour" with a clean, centered layout and gradient background.

## Project Structure
- `index.html` - Main HTML page with embedded CSS styling

## Features
- Centered text display
- Purple gradient background
- Responsive design (mobile compatible)
- Clean typography

## Running the Project
The project uses Python's built-in HTTP server to serve the static HTML file on port 5000.
